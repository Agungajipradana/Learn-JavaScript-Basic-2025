JavaScript Window Methods

The window object is the global object in browser-based JavaScript. It provides a collection of useful methods for interacting with the user and controlling time-based functions.

Common Window Methods

Method Description
alert() Displays a simple alert dialog box with a message.
confirm() Displays a dialog box with OK and Cancel options.
prompt() Displays a dialog box that prompts the user for input.
setTimeout() Executes a function after a specified delay.
setInterval() Repeats a function at specified intervals.
clearTimeout() Stops a function scheduled with setTimeout.
clearInterval() Stops a function scheduled with setInterval.
open() Opens a new browser window or tab.
close() Closes the window opened with window.open().
print() Opens the browserâ€™s print dialog.

Examples


// Alert dialog
alert("Welcome to our website!");

// Confirm dialog
const confirmed = confirm("Do you want to continue?");
if (confirmed) {
  alert("You chose to continue.");
} else {
  alert("You canceled.");
}

// Prompt dialog
const name = prompt("What is your name?");
if (name) {
  alert("Hello, " + name + "!");
}

// Timeout
setTimeout(() => {
  console.log("2 seconds have passed.");
}, 2000);

// Interval
let count = 0;
const intervalId = setInterval(() => {
  console.log("Interval #" + (++count));
  if (count === 3) clearInterval(intervalId);
}, 1000);
